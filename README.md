# Pizza Sales Analysis Report

🚀 **Project Overview**

This project focuses on analyzing pizza sales data to derive actionable business insights. The dashboard provides a comprehensive view of sales performance, helping stakeholders make data-driven decisions.

The analysis includes revenue trends, product performance, sales distribution, and customer ordering patterns.

**Key Metrics**
1. Total Revenue: 817.86K
2. Total Units Sold: 50K
3. Total Orders: 21K
4. Average Order Value: 38.31

**Key Insights**
- 🍕 Top-Selling Pizzas: Thai Chicken Pizza & Barbecue Chicken Pizza generate the highest revenue.
- 📊 Category Analysis: Balanced contribution across Classic, Supreme, Chicken, and Veggie categories.
- 📦 Size-wise Revenue: Large size pizzas contribute the highest share (~46%).
- 📅 Monthly Trends: Peak sales in July; lowest in October.
- 📆 Weekday Trends: Friday records the highest revenue and order volume.

**Tools & Technologies**
- Power BI – Dashboard creation & visualization
- SQL – Data querying & transformation
- Excel – Data preprocessing

**Report Features**
- KPI Cards (Revenue, Orders, Units Sold, AOV)
- Monthly Sales Trend Analysis
- Category-wise Revenue Distribution
- Size-wise Performance Breakdown
- Top Products by Revenue
- Weekday Sales Analysis
